Excel Template for QR Code Bulk Generation

Use this format for your Excel file when uploading to the QR Manager:

Ticket Number Name Phone Email
T001 John Smith 555-0123 john@email.com
T002 Jane Doe 555-0456 jane@email.com
T003 Bob Johnson 555-0789 bob@email.com

Notes:

  • Required: Ticket Number, Name
  • Optional: Phone, Email
  • Save your file as .xlsx format
  • The first row should contain headers
